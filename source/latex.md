title: Add LaTeX equations to your articles.

You can include 

```latex
\LaTeX
```

expressions and equations in your articles. 

## Using LaTeX

To include \LaTeX code in your articles, use the following markdown code:

    ```latex
    \{\frac{4}{5}, \sqrt{49},
    \, 6, \overline{3}, \, 7\sqrt{5}
    \}
    ```

produces:

```latex
\{\frac{4}{5}, \sqrt{49},
\, 6, \overline{3}, \, 7\sqrt{5}
\}
```
